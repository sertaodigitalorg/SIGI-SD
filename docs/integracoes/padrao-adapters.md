# Padrao de adapters

Adapters de integracao devem isolar detalhes de sistemas externos.

Cada adapter deve definir:

- Sistema integrado.
- Finalidade.
- Contrato de entrada.
- Contrato de saida.
- Regras de autenticacao.
- Politica de timeout e retentativa.
- Logs de auditoria.
- Tratamento de dados pessoais.
