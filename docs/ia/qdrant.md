# Qdrant

Qdrant e o banco vetorial usado para armazenar embeddings da base de conhecimento.

Responsabilidades:

- Colecoes vetoriais por dominio.
- Busca semantica para FAQ e documentos.
- Apoio a RAG local.

Qdrant nao substitui o PostgreSQL. Dados relacionais e transacionais permanecem no PostgreSQL.
