# Ollama

Ollama executa modelos locais de IA para tarefas conversacionais do SIGI-SD.

Usos previstos:

- Classificacao de intencao.
- Resumo de atendimento.
- Geracao assistida de respostas.
- Apoio a base de conhecimento com RAG.

As respostas devem ser tratadas como sugestoes operacionais, com controle de seguranca e auditoria.
