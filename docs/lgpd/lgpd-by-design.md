# LGPD by design

O SIGI-SD deve aplicar LGPD desde a concepcao.

Principios:

- Finalidade clara.
- Coleta minima.
- Consentimento quando necessario.
- Retencao definida.
- Auditoria de acesso.
- Anonimizacao quando aplicavel.
- Direito do titular.

Dados sensiveis devem exigir controles adicionais, revisao humana e logs de rastreabilidade.
